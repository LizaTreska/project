## üåü About the Project

**HERITAGE** is a student-led project focused on the digital preservation and 3D reconstruction of Ukrainian cultural heritage sites that have been damaged or destroyed.

This platform serves as a showcase for detailed 3D modeling work, allowing users to interact with digitally restored historical locations and download the source files for educational or archival purposes.

The current featured case is the **National Literary and Memorial Museum of Hryhorii Skovoroda**, which was heavily damaged in May 2022.

## üöÄ Technologies Used

This repository primarily contains the frontend structure of the platform.

* **HTML5:** For the structural markup of the web pages.
* **CSS3:** For styling and design (including modern layouts and responsive design).
* **JavaScript (Vanilla JS):** For client-side interactivity, form handling, and dynamic content loading.
* **VS Code (Editor):** Used for development.
* **Git & GitHub:** For version control.

## üåê Pages / Features

The current platform includes the following key views:

1.  **Home (`index.html`):** Main entry point.
2.  **Case Study (e.g., `item.html`):** Detailed page showcasing a specific 3D reconstruction (like the Skovoroda Museum). Includes history, consequences of damage, 3D modeling process, and download links.
3.  **Gallery (`gallery.html`):** Overview of available digital assets.
4.  **Members (`members.html`):** Information about the project contributors.
5.  **Join Us (`form.html`):** Form for new users or contributors.
6.  **User Profile (`user-profile.html`):** User-specific view.
7.  **Authentication (`login.html`, `regist.html`):** Forms for sign-in and registration (linked to comment/download features).

## üí° Key Features Highlighted

* **Header Navigation:** Clear links to all major sections of the platform.
* **Case Study Presentation:** Rich content featuring text, images, video, and stylized quotes.
* **Download Section:** Dedicated area for downloading `.blend`, `.fbx`, and `.obj` files (requires sign-in).
* **Comment Section:** Interactive section for registered users to leave feedback.
* **Modern Design:** Use of the Inter font and a dark, modern theme for focus.
